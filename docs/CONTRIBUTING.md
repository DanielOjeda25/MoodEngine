# Contribuyendo a MoodEngine

> Este documento es un placeholder. Se completará cuando el proyecto abra contribuciones externas.

## Por ahora

El proyecto está en desarrollo inicial por una sola persona. No se aceptan PRs externos hasta alcanzar un Hito más estable (probablemente tras Hito 10).

Si tenés interés en seguir el proyecto, podés:

- Abrir issues para reportar bugs o sugerir features.
- Seguir el repo para ver commits y releases.

## Cuando se abra a contribuciones

Se documentará aquí:

- Flujo de trabajo (fork, branch, PR).
- Convenciones de commits (ya definidas en `MOODENGINE_CONTEXTO_TECNICO.md` sección 9.8).
- Criterios de revisión.
