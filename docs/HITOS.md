- [x] **F2H12** — CSG operaciones booleanas Subtract / Union / Intersect (completado, tag `v1.3.0-fase2-hito12`). Bloques A-G. Approach destructivo estilo Hammer/TrenchBroom: las ops reemplazan los brushes input por el resultado (1-N brushes convexos), undoable via `BooleanOpCommand` en el `HistoryStack`. **Algoritmo a mano**: plane clipping puro reusando los tipos puros de F2H11 (`Plane`, `Brush`, `BrushFace`, `kPlaneEpsilon=1e-4f`). `subtract(A, B)` itera los planos de B haciendo half-space carving plano por plano (ver Quake/Hammer source); `intersectOp(A, B)` arma un brush con `A.faces ∪ B.faces` validado via `isBrushValid`; `unionOp(A, B)` se compone como `(A subtract B) ∪ {B}` con casos especiales para disjunto / contención. Cardinalidad del resultado: 0 brushes (intersect disjunto, subtract con contención total), 1 (caso simple), N>1 (descomposición convexa). **Bug fix detectado en validación**: los brushes resultantes quedaban con `transform.position=(0,0,0)` y geometría en world space → gizmo en el origen del mundo.
